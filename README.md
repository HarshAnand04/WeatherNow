# 🌦️ WeatherNow — Real-Time Weather Dashboard

## 📌 Overview
WeatherNow is a responsive web application that provides real-time weather updates for any city using the OpenWeatherMap API. It allows users to search for cities and view current weather conditions along with additional details like humidity, wind speed, and forecasts.

---

## 🚀 Features
- 🌍 Search weather by city name
- 📍 Auto-detect user location using Geolocation API
- 🌡️ Display temperature, humidity, and wind speed
- 📅 5-day weather forecast with icons
- 🕘 Search history using Local Storage
- ⚠️ Error handling for invalid city names and network issues
- 📱 Responsive and clean UI

---

## 🛠️ Tech Stack

- OpenWeatherMap API
